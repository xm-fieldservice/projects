# 🚀 快速启动指南

## 一分钟部署

### Windows用户

```cmd
# 双击运行或在命令行执行
deploy.bat
```

### Linux/Mac用户

```bash
# 赋予执行权限并运行
chmod +x deploy.sh
./deploy.sh
```

### 手动启动

```bash
# 1. 安装依赖
npm install

# 2. 启动系统
npm start
```

## 📱 立即访问

启动后访问：**http://localhost:3000**
